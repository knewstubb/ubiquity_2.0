# Form Builder — Requirements (Lightweight)

## Overview
Form creation as a modal overlay from the Journey Builder. Forms capture data that feeds into the contact profile. Supports embedded forms (website) and standalone landing pages.

## Scope
- Modal overlay from journey context
- Field types: text, email, phone, dropdown, checkbox, radio, date, textarea
- Drag-and-drop field reordering
- Field validation rules (required, format, min/max)
- Field mapping to contact attributes
- Form styling: basic theme controls (colours, fonts, button style)
- Submission behaviour: thank you message, redirect URL
- Embed code generation (simulated)
- Progressive profiling (show different fields based on known data — future)

## Sample Data
- Booking enquiry form, newsletter signup, feedback form

## Dependencies
- Journey Builder (form builder opens from journey context)
- Audience Management (form fields map to contact attributes)

## Depends On This
- Analytics (form submission rates)

## Nav Location
- Opened from Journey Builder (modal), also browsable via Content > Forms
