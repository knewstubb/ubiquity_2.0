# Survey Builder — Requirements (Lightweight)

## Overview
Survey creation as a modal overlay from the Journey Builder. Multi-page surveys with question logic and branching. Responses feed into contact profiles and can trigger journey actions.

## Scope
- Modal overlay from journey context
- Question types: single choice, multiple choice, rating scale, NPS, open text, matrix
- Multi-page surveys with page navigation
- Logic branching: show/hide questions based on previous answers
- Survey theming: basic style controls
- Response preview and test mode
- Completion behaviour: thank you page, redirect

## Sample Data
- Customer satisfaction survey, post-treatment feedback, NPS survey

## Dependencies
- Journey Builder (survey opens from journey context)
- Audience Management (responses stored against contact)

## Depends On This
- Analytics (survey response reporting)

## Nav Location
- Opened from Journey Builder (modal), also browsable via Content > Surveys (future)
